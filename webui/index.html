Pixiv

自动上传

高级设置

自动标签会裁到 Pixiv 当前的 10 个上限。手动投稿模式下，浏览器会停在投稿页，方便你确认首张的标题、标签和说明是否符合预期。

LLM 模型支持从提供商自动读取，也支持手动输入。先填写 Base URL 和 API Key，再点读取模型会更顺手。流程上会先做看图打标，再把看图结果和 metadata 一起交给综合整理模型。两个 Prompt 槽位留空时都会回退到内置默认 prompt。

Cookie / CSRF Token 和 LLM API Key 都属于敏感凭证，只建议保存在你自己的本机配置里，不要提交到 GitHub。安全策略默认会自动升到 R-18 / R-18G，并拦截高风险未成年性化组合，喵。

配置改动会自动保存；敏感凭证不会落盘。
